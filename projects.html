Scarlet.gg

A website for organizing tournaments, built in Ruby on Rails and PostgreSQL. Users can register themselves to teams, which can then be registered to tournaments. To be implemented: matches and match pages, and eventually a bracket view.

Basket Case

A winter holiday gift suggestion application that queries the Amazon Product Advertising API, built with Node/Express and MongoDB. This was a group project - my particular contributions were in figuring out how to use the Amazon Node client to work, persisting our items in the database, deploying to Heroku and general refactoring.

MD/VA SSBM Power Ranking

A page that displays the top 20 ranked SSBM players in the MD/VA region. Built using BackboneJS, MongoDB, and Express/Node, mainly as an exercise in creating views with front-end frameworks.
