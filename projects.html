Scarlet.gg: A website for organizing tournaments, built in Ruby on Rails and PostgreSQL. Users can register themselves to teams, which can then be registered to tournaments. To be implemented: matches and match pages, and eventually a bracket view.

Basket Case: A winter holiday gift suggestion application that queries the Amazon Product Advertising API and returns a random basket of items based on user budget, built with Node/Express and MongoDB. This was a group project - my particular contributions were in figuring out how to use the Amazon Node client to work, persisting our items in the database, deploying to Heroku and general refactoring.
